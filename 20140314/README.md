#前端小分队每周例会说明1.0
------
##宗旨

为大家提供一个分享交流的平台。三人行，必有我师！希望能够相互影响，互相督促。形成一个良好的学习交流氛围。使大家在忙于自身业务的同时，能够了解其他业务以及不同场景中遇到的问题。更重要的是快速的学习知识，能够适应互联网的发展节奏。

##愿景
1、每个人每天都能够进步、不管再各个业务中都能够独挡一面。各个成为大牛！！！
2、形成统一高效的前端解决方案
